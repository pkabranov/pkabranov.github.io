Projects

Student Association for Applied Statistics: Data Consulting

Worked with Uizard, a rapidly growing startup that is seeking to revolutionize web and user interface design by tackling the difficulty of finding an image using language. My team developed a sketch-based image retrieval approach using computer vision, deep learning, image processing techniques and Generative Adversarial Networks to analyze a quick freehand sketch and search for similar images to the sketch.

Student Association for Applied Statistics: Data Consulting

I worked with a team of 8 data consultants to help a VC-backed client company, Trace Data, develop an intelligent json key-value pair classification system using word2vec modeling.

Improving Breast Cancer Diagnosis through Machine Learning

In this project, I compared various ML algorithms (KNN, SVM, Logistic Regression, Linear Neural Net) on a 30 feature, 500+ record dataset describing cells from a tumor biopsy (numerical values regarding size, texture, etc.). Proud to have represented Silicon Valley at the Intel International Science Fair in 2018!
